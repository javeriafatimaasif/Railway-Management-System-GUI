# 🚆 Railway Management System GUI

A **Railway Management System GUI** developed using **C# and Windows Forms**.
This project was created as a **university project** to simulate a basic railway ticket booking system with a graphical user interface.

## Project Overview

The system allows users to perform basic railway management operations such as booking tickets, cancelling reservations, and viewing passenger information.
The graphical interface makes the system easy to use and demonstrates fundamental concepts of GUI development in C#.

## Features

 Ticket Booking

 Ticket Cancellation

 Passenger Details Management

 Ticket Fare Calculation

 Basic Ticket Record Handling

 Technologies Used

  C#

.NET Framework

Windows Forms (WinForms)

Visual Studio

## How to Run the Project

Download or clone this repository.

Open the project folder in Visual Studio.

Run the project using the Start button or press F5.
